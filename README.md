# Science_Projects

This repository contains 16 beginner-to-advanced electronics, robotics and IoT projects developed using ESP, Arduino, sensors, motors, and actuators.  
Each project includes its own folder with circuit diagrams, code, components list, and documentation.

- Automatic Dustbin
- Rain Detector System
- Water Level Indicator
- Automatic Traffic & Street Light System
- Automatic Plant Watering System
- Automatic Gate Opening System
- Automatic Tap (IR Based)
- Solar Powered Robot
- Obstacle Avoiding Car
- Alive Human Detecting Robot
- Fire & Smoke Detection System
- Earthquake Rescue Robot
- Multipurpose Agricultural Robot
- Voice Controlled Robot
- Patient Monitoring Robot
- Smart School Bell System
